Exemples Pratiques

Voici quelques patrons Regex utiles pour des cas d'usage courants. Cliquez sur "Tester ce patron" pour le charger dans le testeur ci-dessus avec un exemple.

Adresse Email

Valide la plupart des formats d'emails standards.

[\w.-]+@([\w-]+\.)+[\w-]{2,4}

Téléphone Français

Valide les numéros français avec ou sans séparateurs et préfixe international.

^(?:(?:\+|00)33[\s.-]{0,3}(?:\(0\)[\s.-]{0,3})?|0)[1-9](?:(?:[\s.-]?\d{2}){4}|\d{8})$

Mot de Passe Robuste

Vérifie la présence de majuscule, minuscule, chiffre et caractère spécial (8+ caractères).

^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{8,}$

URL Web

Valide une URL web standard (http ou https).

https?:\/\/(?:www\.)?[-a-zA-Z0-9@:%._\+~#=]{1,256}\.[a-zA-Z0-9()]{1,6}\b(?:[-a-zA-Z0-9()@:%_\+.~#?&\/=]*)

Date (AAAA-MM-JJ)

Valide une date au format AAAA-MM-JJ, en vérifiant les mois et jours de manière basique.

^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$

Code Couleur Hex

Valide un code couleur hexadécimal à 3 ou 6 chiffres, avec un # optionnel.

^#?([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$

Nom d'utilisateur

Valide un nom d'utilisateur de 3 à 16 caractères (lettres, chiffres, underscore, tiret).

^[a-zA-Z0-9_-]{3,16}$

Mots Dupliqués

Trouve les mots identiques qui se suivent immédiatement dans un texte.

\b(\w+)\s+\1\b

Hashtags

Trouve les hashtags dans un texte (ex: #tech, #Python_3).

#[a-zA-Z0-9_]+

Adresse IPv4

Valide une adresse IPv4 standard de 0.0.0.0 à 255.255.255.255.

^((25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)$

Heure (HH:MM)

Valide une heure au format 24h, de 00:00 à 23:59.

^([01]?[0-9]|2[0-3]):[0-5][0-9]$

Code Postal Français

Valide un code postal français standard à 5 chiffres.

^\d{5}$

ID Vidéo YouTube

Extrait l'identifiant unique d'une vidéo depuis différents formats d'URL YouTube.

(?:https?:\/\/)?(?:www\.)?(?:youtube\.com|youtu\.be)\/(?:watch\?v=)?([\w-]{11})

Données de Log

Capture l'adresse IP, la méthode et le statut d'une ligne de log de serveur web commune.

^(\S+) \S+ \S+ \[.*?\] "(\S+) .*?" (\d{3})

Nombre avec Virgules

Valide un nombre entier avec des virgules comme séparateurs de milliers.

^\d{1,3}(,\d{3})*$

Fonctions principales du module re

Le module re de Python fournit plusieurs fonctions pour travailler avec les expressions régulières. Voici les plus importantes. Chaque exemple de code peut être copié en un clic.

Fonction Explication Exemple de code Python
re.search(patron, chaine) Cherche le patron n'importe où dans la chaîne. Renvoie un objet "Match" pour la première occurrence trouvée. 
import re
texte = "J'aime le Python"
match = re.search(r"Python", texte)
if match:
    print(f"Trouvé: {match.group(0)}")
# Output: Trouvé: Python
re.match(patron, chaine) Cherche le patron uniquement au début de la chaîne. 
import re
texte = "Python est super"
match = re.match(r"Python", texte)
if match:
    print("Correspond au début")
# Output: Correspond au début
re.findall(patron, chaine) Trouve toutes les occurrences et les renvoie sous forme de liste de chaînes. 
import re
texte = "chat chapeau chameau"
resultats = re.findall(r"cha[a-z]+", texte)
print(resultats)
# Output: ['chat', 'chapeau', 'chameau']
re.finditer(patron, chaine) Similaire à findall, mais renvoie un itérateur d'objets "Match". Plus efficace pour de grands volumes. 
import re
texte = "Couleur: rouge, Taille: L"
for m in re.finditer(r"(\w+): (\w+)", texte):
    print(f"{m.group(1)} -> {m.group(2)}")
# Output: Couleur -> rouge
#         Taille -> L
re.sub(patron, remp, chaine) Remplace toutes les occurrences du patron par la chaîne de remplacement. 
import re
texte = "Mon numéro est 123-456-7890"
nouveau = re.sub(r"\d", "*", texte)
print(nouveau)
# Output: Mon numéro est ***-***-****
re.split(patron, chaine) Divise la chaîne en utilisant le patron comme délimiteur. 
import re
texte = "Pomme,Banane;Orange"
print(re.split(r"[,;]", texte))
# Output: ['Pomme', 'Banane', 'Orange']
re.compile(patron) Compile une expression régulière pour une réutilisation efficace. 
import re
patron_email = re.compile(r"[\w.-]+@[\w.-]+")
match = patron_email.search("Contact: user@example.com")
if match:
    print(match.group(0))
# Output: user@example.com

Métacaractères et Syntaxe de base

Les métacaractères sont le cœur des expressions régulières. Ils ne représentent pas eux-mêmes, mais ont une signification spéciale.

SyntaxeNomExplicationExemple (Patron)
.PointCorrespond à n'importe quel caractère, sauf une nouvelle ligne.c.t correspond à "cat", "cot", "c&t".
^CirconflexeCorrespond au début de la chaîne.^Début
$DollarCorrespond à la fin de la chaîne.fin$
*AstérisqueCorrespond à 0 ou plusieurs répétitions du caractère précédent.ab*c correspond à "ac", "abc", "abbbc".
+PlusCorrespond à 1 ou plusieurs répétitions du caractère précédent.ab+c correspond à "abc", "abbc".
?Point d'interrogationCorrespond à 0 ou 1 répétition du caractère précédent.couleurs? correspond à "couleur" et "couleurs".
{m,n}AccoladesCorrespond à entre m et n répétitions.a{2,4} correspond à "aa", "aaa", "aaaa".
[...]Classes de caractèresCorrespond à n'importe quel caractère dans les crochets.[aeiou] correspond à n'importe quelle voyelle.
[^...]Classe négativeCorrespond à n'importe quel caractère qui n'est pas dans les crochets.[^0-9] correspond à tout sauf un chiffre.
\|Pipe (OU)Agit comme un opérateur OU logique.chat\|chien correspond à "chat" ou "chien".
(...)Groupe de captureRegroupe une partie du patron et capture le texte correspondant.(https?):// capture "http" ou "https".

Séquences Spéciales

Ces séquences sont des raccourcis pratiques pour des classes de caractères couramment utilisées.

SéquenceExplicationÉquivalent
\dCorrespond à n'importe quel chiffre décimal.[0-9]
\DCorrespond à tout caractère qui n'est pas un chiffre.[^0-9]
\wCorrespond à tout caractère "mot" (alphanumérique + underscore).[a-zA-Z0-9_]
\WCorrespond à tout caractère qui n'est pas un caractère "mot".[^a-zA-Z0-9_]
\sCorrespond à tout caractère d'espacement (espace, tabulation, etc.).[ \t\n\r\f\v]
\SCorrespond à tout caractère qui n'est pas un espacement.[^ \t\n\r\f\v]
\bCorrespond à une frontière de mot (entre \w et \W).N/A
\BCorrespond à une position qui n'est pas une frontière de mot.N/A

Drapeaux (Flags)

Les drapeaux modifient le comportement global de la recherche. En Python, on les passe via l'argument flags (ex: re.IGNORECASE). Dans le testeur ci-dessus, vous pouvez utiliser les équivalents JS (i pour IGNORECASE, g pour global, m pour MULTILINE).

Drapeau (Python)Raccourci (JS)Explication
re.IGNORECASEiIgnore la casse (majuscules/minuscules).
re.MULTILINEmFait en sorte que ^ et $ correspondent au début/fin de chaque ligne.
re.DOTALLsPermet au métacaractère . de correspondre aussi au caractère de nouvelle ligne (\n).
N/A (implicite)gGlobal. Trouve toutes les correspondances, pas seulement la première.
re.VERBOSEN/APermet d'écrire des patrons plus lisibles avec des commentaires (Python uniquement).
