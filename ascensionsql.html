🎒 Préparation de l'équipement (Google Colab)

L'expédition se déroule sur Google Colab : aucune installation, un compte Google suffit — DuckDB permet d'écrire du vrai SQL directement sur des DataFrames pandas. Chaque exercice est une requête à relever dans une cellule.

Exécute d'abord cette cellule : elle charge les 8 tables de la base e-commerce W3Schools depuis GitHub. Ensuite, pour chaque exercice, tu écris ta requête entre les triples guillemets de duckdb.sql(""" ... """).df().

import duckdb
import pandas as pd

base = "https://raw.githubusercontent.com/Patrick-Wampe/datasets/refs/heads/main/"
df_Customers = pd.read_csv(base + "Customers.csv")
df_Products  = pd.read_csv(base + "Products.csv")
df_Orders    = pd.read_csv(base + "Orders.csv")
# … idem pour OrderDetails, Categories, Shippers, Suppliers, Employees

# Pour chaque exercice, écris ta requête entre les triples guillemets :
duckdb.sql("""
    SELECT * FROM df_Customers WHERE City = 'Paris'
""").df()

Les 8 tables et leurs colonnes — garde-les sous la main :

📋 df_Customers · CustomerID, CustomerName, ContactName, Address, City, PostalCode, Country
📦 df_Products · ProductID, ProductName, SupplierID, CategoryID, Unit, Price
🛒 df_Orders · OrderID, CustomerID, EmployeeID, OrderDate, ShipperID
🧾 df_OrderDetails · OrderDetailID, OrderID, ProductID, Quantity
🏷️ df_Categories · CategoryID, CategoryName, Description
🚚 df_Shippers · ShipperID, ShipperName, Phone
🏭 df_Suppliers · SupplierID, SupplierName, ContactName, Address, City, PostalCode, Country, Phone
👥 df_Employees · EmployeeID, LastName, FirstName, BirthDate, Photo, Notes
