🎒 Préparation de l'équipement (Google Colab)

L'expédition se déroule sur Google Colab : aucune installation, un compte Google suffit — scikit-learn y est déjà disponible. Chaque exercice est un défi à relever dans une cellule de code.

Avant de partir, exécute cette cellule : elle charge le Titanic, le rend entièrement numérique et retire les valeurs manquantes. Ce df propre est la base de tous les exercices ; à partir du camp 3, on réutilise aussi le découpage train/test.

import pandas as pd
import seaborn as sns

df = sns.load_dataset("titanic")
df = df[["survived", "pclass", "sex", "age", "sibsp", "parch", "fare"]].copy()
df["sex"] = df["sex"].map({"male": 0, "female": 1})
df = df.dropna()
df.head()

Colonnes retenues : survived (cible), pclass, sex (0/1), age, sibsp, parch, fare — toutes numériques, prêtes pour scikit-learn.
